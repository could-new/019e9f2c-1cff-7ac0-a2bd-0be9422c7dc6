# Panduan Peran Rapat

Aplikasi informatif sederhana yang dibuat dengan Flutter untuk memberikan penjelasan mendetail mengenai peran "Moderator" dan "Notulen" dalam sebuah acara, rapat, atau diskusi.

## Fitur
*   **Beranda Responsif**: Memilih informasi peran yang ingin diketahui.
*   **Informasi Moderator**: Pengertian, tugas, dan fungsi seorang moderator.
*   **Informasi Notulen**: Pengertian, tugas, dan fungsi seorang pencatat/notulis.
*   **Dukungan Lintas Platform**: Berjalan mulus di iOS, Android, Desktop, dan Web.

## Cara Menjalankan
1. Pastikan Anda telah menginstal [Flutter SDK](https://flutter.dev/).
2. Jalankan `flutter pub get` untuk mengunduh dependensi (jika ada).
3. Jalankan aplikasi menggunakan perintah `flutter run`.
